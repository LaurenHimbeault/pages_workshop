## **Step 7: Updating & Maintaining Your Website**

### **Making Changes**
After making changes locally:
1. **Add changes:**
   ```sh
   git add .
   ```
2. **Commit changes:**
   ```sh
   git commit -m "Updated resume"
   ```
3. **Push changes to GitHub:**
   ```sh
   git push origin main
   ```
4. **Refresh your website** to see the updates.
